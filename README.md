CampuSee is a student-exclusive digital platform designed to enhance campus communication and resource sharing by providing a secure, centralized hub where university students can quickly access academic support, find lost items, connect with study partners, and obtain help with daily campus needs, fostering collaboration, reducing academic isolation, and improving the overall university experience.

## Features

- Student Authentication System
- Post and Reply Management System
- Categorized Content Organization
- Search and Filter Functionality
- Real-time Notification System
- Resource Sharing and Exchange Tools
- etc. (future updates)

## Tech Stack

| Layer    | Technology                                    | Purpose / Usage                                                 |
| -------- | --------------------------------------------- | --------------------------------------------------------------- |
| Frontend | [React Native](https://reactnative.dev/)      | Mobile app framework for iOS and Android                        |
|          | [Expo](https://expo.dev/)                     | Development environment, build & deployment tools               |
|          | [TypeScript](https://www.typescriptlang.org/) | Type safety and better code structure                           |
| Backend  | [Supabase](https://supabase.com/)             | Backend-as-a-Service for authentication and database management |

## Prerequisites

| Category        | Tool / Technology         | Purpose / Notes / Installation                                   |
| --------------- | ------------------------- | ---------------------------------------------------------------- |
| System          | Windows 10+ / Android 15+ | Windows 10+ for development, Android 15+ for testing on devices. |
| Node.js         | Node.js                   | JavaScript runtime for React Native & Expo. v18+ recommended.    |
| Package Manager | npm / yarn                | npm comes with Node.js; yarn optional.                           |
| Expo CLI        | Expo CLI                  | For running and building React Native apps.                      |

## Cloning and Initialization

**Step 1:** Clone the repository

```bash
  git clone https://github.com/Jasner13/CampuSee
```

**Step 2:** Install dependencies

```bash
  npm install
```

## Run Locally

**Step 1:** Run expo

```bash
  npx expo start
```

**Step 2:** Using Expo Go, scan the QR code or manually type the URL found in the frontend terminal.
